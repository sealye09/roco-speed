# Roco Speed

洛克王国宠物数据速查工具，速度种族各级强化数据计算，方便卡速度。

## 功能特性

- 🔍 宠物搜索与筛选
  - 按名称搜索
  - 按系别筛选
  - 序号范围筛选
- 📊 属性计算
  - 支持等级、守护兽、努力果配置
  - 性格加成计算
  - 天赋值设置
  - 强化值范围预览
- 📋 表格功能
  - 自定义显示列
  - 分页浏览
  - 多选操作
  - 响应式设计

## 开发

```bash
# 安装依赖
pnpm install

# 启动开发服务器
pnpm dev

# 构建
pnpm build
```
